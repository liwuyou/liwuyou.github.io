实现思路

  1. 图片加载

    • 使用fetch API动态获取assets/images目录下的图片资源
    • 通过DOMParser解析目录结构，提取图片路径
    • 自动生成图片网格布局

  2. 图片展示

    • 使用CSS Grid布局实现响应式图片网格
    • 图片容器保持1:1比例，使用object-fit: cover保持图片比例
    • 添加hover缩放效果提升交互体验

  3. 图片查看

    • 实现模态框查看大图功能
    • 支持左右箭头切换图片
    • 点击背景或关闭按钮关闭模态框

  4. 缩放功能

    • 实现图片区域缩放控制
    • 支持放大、缩小、重置操作
    • 缩放时动态调整网格列数和间距
    • 添加平滑过渡效果

  1. 将图片放入assets/images目录
  2. 打开index.html即可查看相册
  3. 使用右下角按钮控制缩放
  4. 点击图片查看大图，使用左右箭头切换
